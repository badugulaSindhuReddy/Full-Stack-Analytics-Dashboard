# Full-Stack Analytics Dashboard

## Overview
This project is a full-stack analytics dashboard built to visualize and analyze large datasets using interactive charts and filters.

## Tech Stack
- Frontend: React.js
- Backend: Java Spring Boot
- Database: PostgreSQL
- APIs: REST APIs

## Features
- Dynamic dashboards with filters (date, category)
- REST APIs for data retrieval and processing
- Optimized SQL queries for aggregation
- Interactive charts for trend analysis
- Pagination and sorting for large datasets

## Architecture
Frontend (React) → Backend APIs → Database (PostgreSQL)

## Key Highlights
- Designed for scalable, data-heavy applications
- Optimized SQL queries for performance
- Clean API design for efficient data flow

## How to Run
1. Start backend server
2. Run frontend using npm start
3. Connect to database
